洛谷P5019-铺设道路

不得不说比第一次做这道题时要轻松太多了~

题意概括

春春是一名道路工程师，负责铺设一条长度为 n 的道路。

铺设道路的主要工作是填平下陷的地表。整段道路可以看作是 n 块首尾相连的区域，一开始，第 i 块区域下陷的深度为 $d_i$ 。

春春每天可以选择一段连续区间[L,R] ，填充这段区间中的每块区域，让其下陷深度减少 1。在选择区间时，需要保证，区间内的每块区域在填充前下陷深度均不为 0 。

春春希望你能帮他设计一种方案，可以在最短的时间内将整段道路的下陷深度都变为 0 。

题目分析

很容易就能想到要用动态规划（当年在考场上我硬是递归扫描配上一大堆分块啊什么的优化A了这个题…）。单独分析每一块地，他的每一单位下陷深度要么和该地块的前一地块一起减少，要么独立地进行减少。对于第一种情况，我们不用额外增加次数；对于第二种情况，则需要额外的填平次数。

进一步分析，如果前一块地的下陷深度超过了目前的这一块地，我们就可以在填平前一路段的同时也处理好当前路段；如果反过来，则可以让当前路段随着前一路段一起填平相同的深度，然后再独立地填平额外的深度。

思路比较简单。
